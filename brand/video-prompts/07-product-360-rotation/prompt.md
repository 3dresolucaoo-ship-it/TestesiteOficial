# 07 · Catálogo · Produto 3D girando 360°

## Onde usar
- Card de produto no catálogo público Hayzer (`/catalogo/[slug]`)
- Quando o maker subir foto do produto impresso, anima 1x ao hover
- Showcase de portfolios

## Mood
Showcase de produto profissional. Mostra qualidade do produto impresso. Realismo do filamento (linhas de camada, brilho do PETG/PLA).

## Prompt EN

```
Hyperrealistic 3D model of a freshly 3D-printed decorative object rotating slowly 360 degrees, transparent emerald PETG filament showing visible layer lines, dramatic side lighting from rim light, dark gradient background, product showcase style, 6 second loop, no text, glossy surface highlights
```

## Variações

**Variação A — peça funcional:**
```
3D printed mechanical part rotating 360 degrees, matte PLA texture in dark colors, technical product photography style, soft studio light, 6 second loop
```

**Variação B — peça artística:**
```
Decorative 3D printed sculpture rotating 360 degrees, translucent material with internal infill pattern visible, artistic studio lighting, dark background, 8 second loop
```

## Formato alvo
- `.webm` com alpha (sobrepõe em card)
- 720x720 quadrado
- Peso: 800 KB máx
- Duração: 6-8 segundos rotação completa

## Critérios de aprovação
- [ ] Camadas visíveis (mostra que é 3D print real, não CGI plástico)
- [ ] Rotação suave 360°
- [ ] Sem corte visual ao reiniciar loop
- [ ] Fundo neutro ou transparente
