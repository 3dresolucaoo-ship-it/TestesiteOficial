# 03 · Loader · H líquido formando

## Onde usar
- Loading states (entre navegação de páginas internas)
- Splash inicial do app PWA
- Transição em ações longas (gerar relatório, exportar PDF futuro)

## Mood
Minimal, abstrato, breve. Não quer chamar atenção, só ocupar 1-2s sem irritar.

## Prompt EN

```
Abstract 3D liquid metal in emerald and amber colors, swirling slowly in spiral motion, forming and dissolving the letter H, frosted glass texture, dark petrol background, minimalist editorial vibe, 2 second seamless loop, no text, glossy reflections
```

## Variações

**Variação A — só esmeralda:**
```
Liquid emerald metal swirling and forming letter H, then dissolving, dark background, 2 second loop, minimal
```

**Variação B — partículas:**
```
Emerald particles converging in spiral motion to form letter H glowing softly, then exploding back to dust, dark petrol void, 2 second loop
```

## Formato alvo
- `.json` Lottie (preferência) — peso muito menor que vídeo
- OU `.webm` com alpha
- 256x256
- Peso: 50 KB Lottie / 200 KB webm
- Duração: 1.5-2 segundos loop infinito

## Critérios de aprovação
- [ ] Letra H reconhecível
- [ ] Loop sem corte
- [ ] Não cansa em 10 ciclos seguidos
- [ ] Funciona pequeno (24x24px)
