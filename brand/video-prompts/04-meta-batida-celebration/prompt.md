# 04 · Meta Batida · Celebração visual

## Onde usar
- Notificação celebratória quando bate meta financeira do mês
- Animação no card "META BATIDA" quando alcançada
- Pop-up de conquista (lifetime: 100 vendas, 1000 vendas, etc)

## Mood
Celebração brasileira, calorosa, mas sofisticada. Não é confete grátis, é mais "raiz floresceu".

## Prompt EN

```
3D render of glowing emerald roots bursting upward in slow motion, particles of golden ember light scattering, organic explosion frozen mid-motion, dark petrol background, cinematic depth of field, looping 3 seconds, no text, hyperrealistic
```

## Variações

**Variação A — explosão lenta:**
```
Slow motion emerald root system blooming outward like fireworks, golden ember particles, dark cinematic background
```

**Variação B — broto vira flor:**
```
Single emerald root growing rapidly into a stylized flower with golden ember pollen drifting, dark petrol void, hyperrealistic, 3 second loop
```

## Formato alvo
- `.webm` com alpha (sobrepõe em notificação)
- 600x600
- Peso: 400 KB máx
- Duração: 3 segundos (não loop, dispara 1x na celebração)

## Critérios de aprovação
- [ ] Celebra sem ser cafona
- [ ] Sem confete (cliché demais)
- [ ] Cores Hayzer (verde + ember dourado)
- [ ] Tem energia mas não é estressante
