# 01 · Hero · Raiz Hayzer crescendo

## Onde usar
- Hero do dashboard `/dashboard` (vídeo de fundo, opacity 30-50%)
- OU hero principal da landing `hayzer.com.br` (se quisermos trocar o atual)
- OU intro de 3s no load do site

## Mood
Cinematográfico, orgânico, vivo. Mostra a essência da marca Hayzer (raízes verdes em fundo escuro petrol). Sensação: algo nascendo do escuro, crescendo, ganhando força.

## Prompt EN (pra colar na IA de vídeo)

```
Cinematic 3D render of a single emerald-green seedling root growing in slow motion from total darkness, organic tendrils unfurling in spiral motion against deep petrol-teal background, soft volumetric light catching the wet surface of the root, hyperrealistic, 4K, slow camera dolly forward, ambient minimalist mood, no text, looping seamlessly, dark cinematic palette, shallow depth of field, particles of dust catching warm light, 5 second loop
```

## Variações pra testar

**Variação A — câmera estática:**
```
Static cinematic shot of an emerald-green root growing slowly upward against deep petrol-teal void, hyperrealistic, organic motion, soft volumetric backlight, 5 second seamless loop
```

**Variação B — top-down:**
```
Top-down 3D view of intricate root system glowing emerald slowly spreading across dark soil, neural network pattern, particle dust catching warm side light, slow cinematic reveal, 7 second loop
```

**Variação C — close-up macro:**
```
Macro close-up of single tendril of emerald root unfurling in slow motion, wet glossy surface, ember-colored particles drifting past, petrol-teal void background, hyperrealistic, looping
```

## Formato alvo
- `.webm` ou `.mp4 H.264`
- 1920x1080
- Peso: 2-3 MB máx
- Duração: 5-7 segundos loop perfeito (importante: começo e fim costuram)

## Critérios de aprovação (CEO marca ✅)
- [ ] Raiz claramente verde-esmeralda (cor: emerald-400 ou próximo)
- [ ] Fundo petrol-teal escuro (não preto puro, não roxo)
- [ ] Movimento orgânico, não mecânico
- [ ] Loop sem corte visível
- [ ] Sem texto ou marca d'água
- [ ] Sente "vida nascendo" e não "raiz morta de planta velha"
