# 05 · Login · Sistema de raízes amplo

## Onde usar
- Tela de login `/login` (background tela cheia)
- OU hero alternativo da landing
- OU intro de 5s no primeiro acesso

## Mood
Grandioso, sistêmico, conexões. Mostra que Hayzer é REDE (raízes interconectadas = sistemas do maker integrados).

## Prompt EN

```
Top-down 3D view of intricate root system glowing emerald slowly growing across dark soil, branches spreading organically like neural network, particle dust catching warm side light, slow cinematic 5 second reveal, looping, dark petrol-teal background, hyperrealistic
```

## Variações

**Variação A — perspective:**
```
Wide cinematic angle 3D view of emerald root system spreading across dark void, neural network pattern, soft volumetric backlight, slow drift camera, 8 second loop
```

**Variação B — formação progressiva:**
```
Time-lapse of emerald root system growing from single point outward in spiral pattern, particle effects, dark cinematic background, 5 second seamless loop
```

## Formato alvo
- `.webm` ou `.mp4 H.264`
- 1920x1080
- Peso: 2-3 MB máx
- Duração: 5-8 segundos loop

## Critérios de aprovação
- [ ] Sensação de SISTEMA (rede conectada), não raiz solta
- [ ] Cores petrol + emerald
- [ ] Movimento contínuo, não estático
- [ ] Loop sem corte
